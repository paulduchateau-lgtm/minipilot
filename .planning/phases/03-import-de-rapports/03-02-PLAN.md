- Use `useNavigate` from react-router-dom

**Step 1 — Upload:**
- Title: "Importer un modele de rapport" (Source Serif 4, fontSize 28, fontWeight 300)
- Subtitle: "Deposez un fichier Excel (.xlsx) ou Word (.docx) pour recreer un rapport equivalent." (DM Sans 14px, muted)
- Drop zone: dashed border (2px dashed var(--mp-border)), borderRadius var(--radius-md), padding 48px, text-align center, cursor pointer
- Drop zone supports drag-and-drop (onDragOver preventDefault, onDrop get files[0]) AND click (hidden input type="file" accept=".xlsx,.docx,.xls,.doc")
- On file selected: validate extension is .xlsx/.xls/.docx/.doc, set step=2, call handleUpload(file)
- Show Upload icon from lucide-react in drop zone
- If error state set, show error message in red (var(--mp-danger) or #C45A32)

**Step 2 — Analyzing:**
- Centered layout with Loader2 spinning icon
- Text: "Analyse du modele en cours..." (DM Sans 14px, muted)
- Sub-text: "L'IA identifie la structure, les KPIs et les champs de donnees." (DM Sans 13px, tertiary)
- This step auto-advances when handleUpload completes

**handleUpload(file):**
```
try {
  setError(null);
  const result = await api.importTemplate(file);
  setFingerprint(result.fingerprint);
  setAvailableColumns(result.availableColumns);
  setStep(3);
} catch (err) {
  setError(err.message || "Erreur lors de l'analyse du modele.");
  setStep(1);
}
```

**Step 3 — Field Mapping:**
- Title: fingerprint.title (Source Serif 4, 24px, fontWeight 400)
- If fingerprint.objective: show it as subtitle (DM Sans 14px, muted, marginBottom 20)
- If fingerprint.kpis.length > 0: show a "KPIs detectes" section with badges (IBM Plex Mono 10px UPPERCASE, pill shape with accent dot)
- If fingerprint.sections.length > 0: show a "Sections detectees" summary — list section titles with type badge (e.g., "bar", "table", "text")
- Then show TemplateFieldMapper with detectedFields=fingerprint.detectedFields, availableColumns, loading=generating
- onConfirm handler: builds suggestion object from fingerprint + mapping, calls api.generateReport(suggestion), on success navigates to `/${slug}/report/${report.id}`, on error sets error message

**handleConfirm(mapping):**
```
setGenerating(true);
try {
  const suggestion = {
    title: fingerprint.title,
    description: fingerprint.objective || "Rapport importe depuis un modele",
    type: fingerprint.sections?.[0]?.type || "bar",
    columns: Object.values(mapping).filter(Boolean),
    kpis: fingerprint.kpis?.map(k => k.label) || [],
    templateFingerprint: fingerprint,
    columnMapping: mapping,
  };
  const result = await api.generateReport(suggestion);
  navigate(`/${slug}/report/${result.report.id}`);
} catch (err) {
  setError(err.message || "Erreur lors de la generation du rapport.");
  setGenerating(false);
}
```

- "Retour" link at top of all steps: navigates to `/${slug}/reports` (DM Sans 13px, muted, flex with left arrow)
- Max-width: 680px, centered with margin 0 auto, padding 32px
