   - Renders editable fields:
     a. Title: text input bound to `section.title`
     b. Insight: textarea bound to `section.insight`
     c. X Axis Key: text input bound to `section.config.xKey`

     e. Data rows: simple table with "Add row" button. Each row is an object. Column headers come from xKey + yKeys. Each cell is an editable input. "Delete row" button per row.

   - On any field change, call `onChange({ ...section, [field]: newValue })` (immutable update)
   - For data rows, parse numeric values with `parseFloat` when possible, keep strings otherwise
